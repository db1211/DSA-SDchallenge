### Problem 1: Two Sum (Brute Force)
```
Given an array of integers nums and an integer target,
return the INDICES of the two numbers that add up to target.

You may assume each input has exactly one solution.

Example 1:
Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9

Example 2:
Input: nums = [3, 2, 4], target = 6
Output: [1, 2]
Explanation: nums[1] + nums[2] = 2 + 4 = 6
```

**Your Task:** Write `twoSum(int[] nums, int target)` that returns `int[]`

**Hint for Brute Force:** Check every pair using two loops.

### Problem 2: Contains Duplicate
```
Given an integer array nums, return TRUE if any value appears 
at least twice in the array. Return FALSE if every element is distinct.

Example 1:
Input: nums = [1, 2, 3, 1]
Output: true

Example 2:
Input: nums = [1, 2, 3, 4]
Output: false

Example 3:
Input: nums = [1, 1, 1, 3, 3, 4, 3, 2, 4, 2]
Output: true
