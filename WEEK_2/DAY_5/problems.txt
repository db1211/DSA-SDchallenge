import java.util.HashSet;

HashSet<Integer> set = new HashSet<>();
set.add(5);              // add element
set.contains(5);         // check exists → true
set.remove(5);           // delete
set.size();              // count elements
```

Key difference: **HashSet only stores unique values, no duplicates!**
```
